# EcoWatt Monitor

## Visão Geral

O **EcoWatt Monitor** é uma plataforma web de monitoramento de consumo energético em tempo real, projetada para auxiliar residências e pequenos comércios a otimizarem seu uso de eletricidade. Alinhado ao **Objetivo de Desenvolvimento Sustentável (ODS) 7 da ONU - Energia Limpa e Acessível**, o projeto visa combater o desperdício de energia através da disponibilização de dados acionáveis e insights sobre os padrões de consumo.

A solução permite que os usuários cadastrem seus dispositivos, visualizem dashboards interativos e recebam relatórios detalhados, promovendo uma cultura de eficiência e sustentabilidade energética.
